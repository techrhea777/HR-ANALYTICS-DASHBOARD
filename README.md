# 📊 HR Analytics Dashboard

## Overview
This project presents an **HR Analytics Dashboard** created from an HR dataset of about **1500 rows**.  
The dashboard helps analyze employee data and provides insights into attrition, employee demographics, job satisfaction, and overall workforce trends.

## Dashboard Preview
![HR Analytics Dashboard](HR%20ANALYTICS.png)

## Dataset Information
- **Dataset Type:** HR Analytics
- **Total Rows:** 1500
- **File Format:** Excel / CSV
- **Main Areas Covered:**
  - Employee Age
  - Gender
  - Department
  - Job Role
  - Education
  - Attrition
  - Job Satisfaction

## Key Metrics
- **Overall Employees:** 1470
- **Attrition:** 237
- **Attrition Rate:** 16.12%
- **Active Employees:** 1233
- **Average Age:** 37

## Dashboard Insights
The dashboard includes:
- Department-wise attrition analysis
- Employee count by age group
- Job satisfaction rating by role
- Education field-wise attrition
- Attrition rate by gender for different age groups

## Tools Used
- **Power BI** for dashboard creation
- **Excel / CSV** for dataset storage and cleaning
- **GitHub** for project sharing and version control

## Project Objective
The purpose of this project is to:
- Analyze employee attrition trends
- Understand workforce demographics
- Identify patterns in job satisfaction
- Support HR decision-making with data insights
